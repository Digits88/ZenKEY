ZenKEY Configuration - Adding or editing Items


When you add or edit an item, you are presented with the following dialog box.

The item properties have the following effect:

  • Caption - The text that appears in the ZenKEY menu for this item
  • Action type - The action category that describes this action. Different actions will allow you to choose different parameters. Here, when choosing an action to open a file or run a program, you must specify which file to open.
  • Hotkeys - Choose the key combination that will fire the action
  • Set - Click here and press the key combination you want to use and ZenKEY will detect your keypresses and set them accordingly.
