ZenKEY Configuration - Settings


When you choose the settings tab, you will see the various options available to you..

The Appearance options are as follows:

  • Skin - Choose which graphics to use for the ZenKEY form, systemtray icon and menu icons.
  • Transparency level - This determines the level of transparency applied to the ZenKEY window and the 'Desktop Map'.
  • No icons- Use no icons in the ZenKEY menu.
  • Zen icons - Use a recursive rotating icon in the menu.
  • Load file name icons - Display an icon in the menu which corresponds to the first letter of the caption.

The Behavior options are as follows:

  • Follow active window - Tells the ZenKEY window to attach itself to whichever window you are using. ZenKEY will then move with the Window whenever you move it via a ZenKEY 'Window action'.
  • Hide tooltip text - Prevents the ZenKEY Configuration Utility from displaying tooltip help whenever you move over an item.
  • Always show form - Make the ZenKEY window visible or invisible.
  • Load on startup - Load ZenKEY automatically whenever windows starts.
  • Layer (z order) - This setting specifies the depth at which the ZenKEY windows are displayed relative to other windows (z order) - bottommost, normally, on top or topmost.

The Window action options are as follows:

  • Act on window under mouse (else active) - Windows actions are carried out on the window you are currently using. If you do not enable this setting, the actions are performed on whatever window lies directly underneath your mouse cursor at the moment the action is fired.
  • Remember the last X window actions - This setting determines how man window actions you can undo.
  • When moving window, shift X pixels - This setting determines the magnitude of the window move and shift actions. Please note that the height is altered by fewer pixels in order to maintain the windows' proportions(i.e. aspect ratio).

The Quotes and messages options are as follows:

  • Show quotes - Forces quotes to be displayed alongside all ZenKEY messages. Just to keep life interesting...
  • From beginning - Resets ZenKEY tips and guides to be displayed from the beginning.
  • Quote group - Choose the types of quotes you would like to be displayed. Wise words, jokes, one-liners. Whatever lights your wick.
  • Show startup tips and quotes - Show a quote whenever ZenKEY starts.

The 'Auto-window transparency' settings allow you to make windows transparent or opaque as you move between them. You can set the level of transparency for active windows (windows you are using) and inactive windows (windows that you are no longer using). The settings are follows:

  • Enable Auto-window transparency - Enables or disable this feature.
  • "Setting" when the window is active - This determines the level of transparency for windows that are active or in-use.
  • or was one of the last "X" active windows - This setting allows you to specify how many of the windows you have last used should be regarded as active eg. a setting of 2 will keep the last 2 windows you have used at the Active window transparency level.
  • "Setting" when the window is inactive - This determines the level of transparency for windows that are inactive or not in-use.
  • Do not apply transparency to these programs - Transparency can often interfere with program that maintain tight control over their windows or that set their own transparency levels. Using the 'Add' and 'Remove' buttons here, you can specify any programs that should be left alone and ignored when applying transparency.
