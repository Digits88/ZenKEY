Window selection


Window actions are normally applied to the active window or the window under the mouse, depending on your chosen setting. You can, however, dynamically create a selection of windows all of which will have that action applied to them.

For example, if you have 3 windows selected, firing the 'Move window left' action will cause all 3 windows to be moved left. Please note that this only applies to actions that pertain to single windows: it makes to sense to apply the 'Minimize all windows' action to 3 windows.

You can add windows to the selected window list by:

  • holding down 'Control' and clicking on them in the 'Desktop map'
  • selecting a window, then poping up a ZenKEY menu with 'Window selection' actions in it.
  • assigning 'Window selection' actions to hotkeys and pressing these keys to apply them to the active window.

Note: Although window selection functions work most easily with the 'Desktop map' enabled, they will still work without it.
