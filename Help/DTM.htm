The 'Desktop map'

Designed to work in collaboration with its sister feature, the 'Infinite desktop', the 'Desktop map' is a small window that displays the relative locations of all visible windows. Using this map, you can keep track of where windows have been placed, which one is active and where the desktop is currently positioned.

The blue rectangle represents your current desktop area, the yellow rectangle represents your 'active' application window, and the green rectangles show the remaining visible windows.

 

 

 

 

 

 


How it works

The desktop map displays a scaled down representation of all visible program windows. It is designed to help you manage your program windows when the 'Infinite desktop' feature is enabled, but it can be used independently.

You can move windows by dragging their rectangles on the map, or simply activate them by clicking inside their corresponding rectangle. Holding down 'Control' will allow you to select multiple rectangles/windows at the same time. Clicking on the map itself (outside of the rectangles) will deselect any selected windows. Clicking on the green square will allow you to alter the ZenKEY transparency or show the configuration utility, whilst clicking on the red square will close the map.

Right-clicking in a rectangle (or outside one if you have used 'Control' to select them) will pop-up a list of actions you can perform on that window(s). Holding down shift or control whilst right-clicking will bring up the 'Move window' and 'Alter window' menus.

You can change the way it behaves and appears via the ZenKEY configuration utility.

Settings

The transparency level of ZenKEY windows can be set from the desktop menu (see illustration above), and determines how "see though" the map and the main ZenKEY window is. Note that: the setting change here is only temporary: to set it permanently, use the ZenKEY Configuration utility.

The remaining settings are available via the Configuration Utility:

  • Show the desktop map XX - This option sets whether or not the desktop map is enabled, and if it is, whereabouts it will appear.
  • As a caption, use XX - This option selects how the window rectangles are labeled on the desktop map. They labels can use the name of the 'exe' file, the window caption or remain empty.
  • Refresh map XX - This option decides how often the desktop map is redrawn. You can choose to have it redrawn only when a program window is activated, only when ZenKEY actions are fired, or when either of these events occur.
    Note : Remember that if a window is dragged, the map will not pick up this movement until you click on it or change applications. We have chosen not to monitor these movements as it would require more processing time and therefore more CPU usage.
  • Show XX YY - This option selects the length of time the map should be displayed for. The second option here decides how the map is displayed: it can be on top of other windows, on the same level, or below them.
  • Window colors - Here you can choose what color to use for the window rectangles on the map. You can choose the color of the active window, the on-screen windows, the off-screen windows, the desktop window and the map background.
