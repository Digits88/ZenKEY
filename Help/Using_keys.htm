Using keys to fire actions

Actions can do many things: launch programs, move windows etc. What gives them more power is that they work from anywhere. This allows you to do things without switching between programs or windows.

Note 1: Remember, that certain keystrokes are used by programs for special functions, such as 'Control + C" for copy, 'Control + V' for paste etc.. Assigning these keystrokes to ZenKEY functions can have frustrating consequences. Be careful what keystrokes you use!

Note 2: Although ZenKEY allows you to assign a single key for an action, be especially careful here, as that means that ONLY ZenKEY will respond to that keystroke!

Note 3: If ZenKEY gives you a message on startup saying that it is 'unable to create a Hotkey', this means that it is being used by another program. Either change the Hotkey in ZenKEY or change it in the other program (to find out what the other program is, just press the Hotkey combination).
