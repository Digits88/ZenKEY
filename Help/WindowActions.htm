Window actions


Window actions change the size and properties of a window or windows. You can tell ZenKEY whether these actions should be performed on the currently active window, the window directly under the mouse pointer or the currently selected windows. There are many possible actions:

  • Minimize all windows - Minimizes all the windows which are not already minimized.
    NOTE : Not all Windows minimize to the taskbar, but insist on becoming nasty little floating boxes!
  • Minimize all windows except active - Minimizes all the other windows except the one you are working on
  • Restore all windows - Restores all the windows which are currently minimized
  • Show Desktop - Go to your current desktop instantly
  • Center window - Places you current window in the center of the screen
  • Maximize window horizontally/vertically - Maximizes the width or height of your window to the maximum that the screen will allow
  • Maximize/Minimize active window - Maximizes or minimizes the window
  • Normalize window - Sets the window to be neither maximized nor minimized
  • Move window up/down/left/right - Moves the selected window by a specified number of pixels
  • Size window to top/bottom/left/right half - Resizes the window to the maximum width/height of the screen in one dimension and half of the screens height/width in the other dimension
  • Fatten/Thin/Heighten/Shorten window- Resizes the selected window by a specified number of pixels (preserving the aspect ratio)
  • Set window to 'Always on top' - Sets the window to be always visible and floating above your active window
  • Remove Always on top - Sets the window to be displayed on top of or beneath other windows as you move between them
  • Disable active window - Prevents the window from receiving any user events, be they clicks, drags or keypresses
    NOTE: This action can be dangerous as you can no longer do anything with that window. For this reason, the window is re-enabled immediately any other Window action is performed and the window action is canceled
  • Drop window to bottom - Sets the window to appear beneath all other windows
  • Enable window - Enable a window (counters 'Disable active window')
  • Kill window - Closes the selected window or application.
  • Send window to tray - Hides the selected window, and places an icon in the system tray (usually bottom right of screen). When you click on this icon, the window will reappear
  • Restore window from tray - Makes the last window to be sent to the system tray visible again.
  • Window Transparency - Sets the window to be drawn with a varying degree of opacity
    NOTE: This works only on Windows XP/2000 and is a performance hog (unless you have a dedicated GPU (Graphics Processing Unit))
  • Remove transparency - Counters 'Window transparency'
  • Undo last window action - This reverses the effect of your last window action. For example, if you last moved a window right, it will be moved back to the left again.
