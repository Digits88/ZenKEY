Iconify Action

An Iconify action hides the active or selected windows and replaces them with small 'icons'. Clicking on these icons will restore the program window/windows to its/their original state. Note that these icons are different from normal icons, which would normally start a new instance of the program. ZenKEY can also display CPU and Memory information on the icon.



Icons can be displayed in two modes - free-floating or locked in a ZenBar layout (selectable in the Configuration Utility). In free-floating mode, you can place icons individually, size them individually, and choose whether or not they should display program information. In ZenBar mode, they are placed and sized automatically, and always display program information.

By right-clicking on the icon, you can specify the following options (n/a. ZenBar indicates that this option is not available in ZenBar mode) :
  • Keep Icon visible - If selected, Icons will remain visible after a window has been restored or the Icon has been clicked.
  • Minimize window (vs. Hide) - When you Iconize a window, it is usually just hidden. This can interfere with some windows that manage their own window state and visibility. Selecting this option usually solves these problems.
  • Preserve position - Save the Icon's position (n/a. ZenBar)
  • Show process information - The owner programs total CPU time and current memory usage will be displayed in the icon. (n/a. ZenBar)
Additional items in the right-click menu include:
  • Close - Remove the Icon.
  • Close and clear settings - Removes the icons and resets all of its settings.
  • Close Icons and applications - Removes the Icon and closes any program windows associated with it.
The right-click menu also contains an 'All icons' menu which performs useful actions that effect all icons.
  • Re-acquire Window - When an application has been Iconized and it is subsequently closed and opened again, ZenKEY will no longer display CPU and Memory information as the recorded window is invalid. Clicking on this option will allow ZenKEY to hunt for the new instances and display their information.
  • Place in ZenBar positions - This allows users of the free-floating mode to place icons in a ZenBar layout once off.
  • Flush dead icons - Removes icons which refer to windows which have since been closed.
