Auto-window transparency

Auto-window transparency is a rather nifty feature that tracks which windows you are using, and automatically sets their level of transparency as you move between them.

ZenKEY maintains a list of the last window you have used, and classifies these as 'Active windows'. You can set how transparent these windows should be made and how transparent the remaining 'Inactive windows' should be made.

You can also specify how many windows should be kept in the Active window set. This way, you can keep the last few windows you have used opaque, making it easier to see information in the last few windows you have used.

There are a few things to keep in mind, however.

  1. Transparency is a real performance hog. A good graphics card is almost essential for this feature to work nicely.
  2. Some program windows, especially video/media applications, do not display properly whilst semi-transparent. Add them to the 'Do not apply...' list.
  3. There is an action that undoes the last window transparency if you ever need to quickly see what was in the last window made transparent.
