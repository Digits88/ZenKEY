The 'Infinite desktop'

Imagine that your desktop is merely a small part of an area that extends upwards, downwards, to the right of and to the left of your screen boundaries. The 'Infinite desktop' feature allows you to use this space, permitting windows to be moved anywhere, even off-screen. Why would you want to move windows off-screen, you ask?

In short, it allows you to create an (almost) infinite desktop space, where groups of related applications can be placed together. Simply moving to this location makes all of the applications in that group visible simultaneously. If you suddenly want to start something else, press a key and jump to a new area. Your desktop space is cleared, and you can get back to your previous programs merely by jumping back!

Together with its sister feature, the 'Desktop map', the 'Infinite desktop' allows you to create a seamless, unlimited application workspace. Pressing a keyboard combination or clicking on the desktop map allows you to jump quickly to any location where the group of related applications lie.

 


How it works

The 'Infinite desktop' works by allowing normal ZenKEY 'window' actions to move windows off the screen. Below is example of this. On the right, you can see the desktop map. The blue rectangle represents your current desktop area, and the red rectangle shows a window that has been moved up and off the screen. If we want to access this application, we simply move one desktop up.

In this way, you can thus arrange and place windows across an unlimited desktop space. In order to navigate and manipulate your unlimited desktop, we introduce a new set of actions. Some of these actions are:

  • Move one desktop up/down/left/right - Moves all your program windows up/down/left/right by a distance equal to your desktop height or width (depending on your direction of movement). These actions allow you to quickly navigate your desktop space.
  • Move half a desktop up/down/left/right - The same as above, but moves are only half of your desktop height or width.
  • Bring window onto desktop - If the active application window is off-screen, this action brings the window back onto your desktop.
  • Center desktop on the active window - Moves the desktop so that the active application window lies at the center of your screen.
  • Return to original desktop position - Restores your desktop position to where you started from. This has the effect of undoing all desktop moves.

Settings

These settings can be accessed via the ZenKEY Configuration Utility.

  • Enable the infinite desktop - If this feature is not enabled, all your program windows will be kept on the screen (the default 'Windows' behavior). When enabled, windows can be moved anywhere....Freedom.
  • Always center on active window - Enabling this feature will tell ZenKEY to always keep your desktop centered on whichever application gets the focus (i.e. is active). This means all program windows will be moved whenever the focus changes. This feature can be quite disorienting, but it is fun nonetheless.
  • Move all windows onto desktop on exit - Using the Infinite desktop means that some windows might lie off-screen. When you exit ZenKEY, you will therefore not be able to access them. Enabling this option forces ZenKEY to move all the windows back onto the desktop whenever ZenKEY closes, thus preventing windows from becoming inaccessible.
  • Do not apply desktop moves to these programs - There might be certain windows that you always want to keep visible i.e. on your desktop. Add these applications to the list shown here, and they will always be kept where they are.
