Simulate a series of predefined Keystrokes
You can define any series of keystrokes which ZenKEY can 'simulate'. This can save time by replacing many keystrokes with one. You can use this feature to:
  • Type out greetings, signatures, disclaimers or any other commonly used text
  • Automate regularly performed tasks
  • Enter commonly used passwords (WARNING : Whilst ZenKEY does encrypt these keystrokes, it is not military strength encryption. Also, anyone using ZenKEY can access them, so please be careful)
  • Create 'Scripts' to view or change program or system settings

In fact, anything you can do via the keyboard, you do via these actions. You can record or create a 'Keystroke' sequence using the following form in the 'ZenKEY Configuration Utility'. Select 'Actions->New->Action type->Simulate a series of Keystrokes->Edit Keystrokes''.

The left side shows you which keys have been recorded so far, whilst the recording panel on the right provides the keystroke recording tools.

Current key state - Displays the state of the recorded keys
Hold keys - Prevents the currently held keys from bieng released when you release them. Using this option, you can simulate a combination of keys without actually pressing the combination (you can thus , for example, record 'Control + Alt + Del' without bringing up the task manager)
Release keys - Releases any keys that ZenKEY has recorded as being down.
Tab, Shift, Control, Windows, Alt - Allows you to simulate the pressing of these keys without actually pressing them (you can thus , for example, record 'Control + Alt + Del' without bringing up the task manager)
Allow depressed keys to fire repeated keypresses - Normally, when you hold a key down, windows interprets this as pressing the key repeatedly (holding down 'z' in a word processor will result in 'zzzzzzz....'). Enabling this option will restore this behavior and prevent the record only single keystrokes.
Start recording - Start/Resume the process of recording your key sequence. If you have already recorded a keystroke sequence, you will be given the option of adding keystrokes to the sequence or inserting them before the currently selected keystroke.
Clear all - Clears all recorded keystrokes, and begins a new key sequence
Remove last - Remove the last keyup and keydown combination (all key sequences must have the same number of 'Keydowns' and 'Keyups' for each key. Items are thus removed in pairs.)
