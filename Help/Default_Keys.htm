Default Hotkeys

Greetings brave fellow / sheela

You have chosen to enable the default hotkeys for ZenKEY (Gasp!). Instead of trying to remember each Hotkey, it is sometimes easier to use a general rule for the different types of actions. For example

'Control + a key' for a Window action
'Control + Alt + a key' for a File/Program
'Alt + a key' for a Winamp action (or compatibles, such as QCD Player, STP etc.)

If in doubt, 'Alt + Space' and 'Control + Space' shows a menu with the actions and their associated hotkeys. Remember, you can add, change or remove any Hotkeys or Actions via the Configuration utility.

Play and enjoy.
